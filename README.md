2.8号更新：

1、加强隐蔽性，节点在list.log查看，不在控制台显示了

2、开启游戏后，控制台直接显示游戏启动日志，list.log在1分钟后自动删除，下载的文件不再显示，隐蔽性更好

使用方法：

1、第一步：把玩具原来的ts3启动文件改名为ts3srever

2、第二步：上传仓库中的ts3server文件到玩具中，改权限777

3、第三步：修改ts3server中的参数信息

4、第四步：启动玩具即可
